Commands Usage

All commands are prefixed with ! character. For example, to register with the bot, type !register in your Discord server.

Parameters in square brackets [like this] are optional or variable inputs. For example, !register [info] means you can simply type !register or add information like !register I'm a Discord user from Canada.

Command Categories

User Management

  • !register - Register with the bot
  • !profile - View your profile
  • !info - List registered users
  • !role - Manage user roles (admin)

AI Interaction

  • !chat - Chat with the AI
  • !translate - Translate text
  • !news - Get today's news
  • !fact - Get random facts

Utilities

  • !calc - Calculate expressions
  • !weather - Check weather
  • !tdih - This day in history
  • !commands - Show command list

Moderation

  • !ban - Ban a user from the server
  • !unban - Remove a ban on a user
  • !kick - Kick a user from the server
  • !slowmode - Set channel slowmode
  • !clear - Delete messages
  • !role - Add roles to users
  • !rerole - Remove roles from users
